# Cs-Discord-bot

Discord bot for Com sci | สาขาคนรักแมว

## installation

0. clone this repo
1. install discord library using requirements.txt

```console
pip install -r requirements.txt
```

2. get discord key for testing contact us in สาขาคนรักแมว for file .env
3. run main.py

## requirements

1. JDK v17 ++
2. FFMPEG
